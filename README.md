# Classification-in-Transformed-Domain
This repository contains various cases in which classification problems are handled by using frequency as feature.
The Classification of MNIST and EMNIST if not specified is done using union mask method

MNIST Input: https://www.kaggle.com/datasets/hojjatk/mnist-dataset

BreakHis Histopathological Database: https://web.inf.ufpr.br/vri/databases/breast-cancer-histopathological-database-breakhis/
